function scope_widget (   owner)

Main fuction to create the oscilloscope widget. Creates the display, and the auxiliary buttons, then connects timer events to refreshers, for displaying the measurements plot.

Parameters
ownera jQueryfied HTML element which will contain the scope
configboolean; must be true to take in account next parameters
inputExpeye's input channel which will be scoped. defaults to "A1"
samplesnumber of measurements : the total duration will be samples * (number-1) seconds ; defaults to 201
delaydelay between two consecutive measurements (in second) defaults to 0.0002 (200 microseconds)
durationtotal duration of scoped data. Defaults to null. if set to some float value, its value takes precedence onto delay and delay will be set to duration/(number - 1).
